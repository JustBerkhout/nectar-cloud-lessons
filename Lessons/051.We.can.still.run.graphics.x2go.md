1. a nectar instance:
  * running x2go server 
1. your computer:
  * running x2go client 
  * running an X server

# Setup the x2go client
1. start x2go
1. Session > New Session
1. Session Name: `x2go test`
1. Host: `your_instance_ip_address`
1. Login: `ubuntu`
1. Try auto login: `enabled`
(or you may need to browse to your ssh key)
